git操作

$git init -- 创建仓库

$git config --global user.name "xiaoming"

$git add ./readme.md` 所指定的文件放到大门口

$git commit -m "这是对这次添加的东西的说明"

$git status

$git log --oneline

$git reset --hard [版本号]

$git reflog -- 查找记录找回版本

在vscode使用git可视化

git分支操作

$git branch [分支名称,dev] -- 创建分支

$git checkout dev -- 切换分支

$git merge dev -- 合并分支.注意：合并分支，需要切换主分支再合并
